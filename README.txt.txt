####### REQUISITOS #########

Ter instalado o Postman na máquina e utilizar o JavaScript (opcional) caso queira alterar a automação.

####### Como Executar #########

Após instalar o Postman, importar o arquivo Json (API) para o Postman;
Depois ir no canto inferior direito em Runner;
Importar os endpoints da API para esteira de automação;
E clicar no botão laranja para executar a automação.

OBS.: A autoação só irá funcionar com um CPF sem restrições.
